# fanuc_r2000ic_description

This package contains a URDF description for the Fanuc R-2000iC/165F manipulator.

## Overview

The original Fanuc R-2000iC description is published in Xacro under the BSD 3-Clause license in the [fanuc](https://github.com/ros-industrial/fanuc) repository, which is part of the [ROS-Industrial](http://wiki.ros.org/Industrial) program. The Xacro was converted to a standalone URDF for use with rigid-transform-kit. See the main [fanuc](https://wiki.ros.org/fanuc) page on the ROS wiki for more information on usage.

## Specifications

| Spec | Value |
|------|-------|
| Payload | 165 kg |
| Reach | 2655 mm |
| Axes | 6 |
| Repeatability | ±0.05 mm |

## Reporting issues

This description is distributed as-is in the hope it can be useful. If you face an issue with the URDF file itself, open an issue here. Please report any other issue to the upstream repository.
